LESSONS

Lesson 1

第1講　文字式と同類項

数学Iの入口でまず身につけるべきことは、式をただ計算することではありません。 式を「部品」に分け、何が同じ種類で、何が違う種類なのかを読む力です。

Question

今日の問い

なぜ 2x + 3x は 5x にまとめられるのに、 2x + 3y はまとめられないのでしょうか。

2x + 3x = 5x
2x + 3y はそのまま

この違いが分かると、文字式の整理、展開、因数分解、方程式のすべてが見えやすくなります。 第1講の目標は、計算を速くすることではなく、まず式を正しく読むことです。

Goal

この講義の到達目標

  • 式を「項」に分けられる。
  • 係数と定数項を見分けられる。
  • 同類項かどうかを判断できる。
  • x と x2 を別物として扱える。
  • 符号を項の一部として読める。
  • 同類項だけを正しくまとめられる。
Definition

まず、式を部品に分ける

次の式を見てください。

3x + 5 - 2x + 4

この式は、次の4つの部品に分けて読むことができます。

項

+ や - で区切られた式の部品。
例：3x、5、-2x、4

係数

文字にかかっている数字。
例：3x の係数は 3。-2x の係数は -2。

定数項

文字を含まない数字だけの項。
例：5、4

同類項

文字の部分が完全に同じ項。
例：3x と -2x は同類項。

Rule

同類項とは何か

同類項とは、文字の部分が完全に同じ項のことです。 ただ「同じ文字が入っている」だけでは足りません。

ax + bx = (a + b)x
まとめられる

2x + 3x = 5x
文字の部分がどちらも x だからです。

まとめられない

2x + 3y
x と y は違う文字なので、そのままです。

これも別物

x + x2
x と x2 は形が違うので、まとめません。

定数項どうし

5 + 4 = 9
数字だけの項どうしはまとめられます。

Why

なぜ同類項だけまとめられるのか

2x は「x が2個」、3x は「x が3個」という意味です。 だから、2x + 3x は「x が5個」と読めます。

2x + 3x = 5x

しかし、2x + 3y は「x が2個」と「y が3個」です。 これは種類が違います。 りんご2個と鉛筆3本を、ただ「5個」とまとめると何の数か分からなくなるのと同じです。

ゼド先生とミアの補講 MINI LESSON
ミア

x と x2 って、どっちも x が入っているから、まとめてもいいのかと思っていました……。

ゼド先生

違う。見るべきなのは、同じ文字が入っているかではない。 文字の形が完全に同じかどうかだ。x と x2 は別物として扱え。

Sign

符号は項にくっついている

文字式でかなり多いミスが、符号を項から切り離してしまうことです。

5x + 8 - 2x + 7

この式の項は、次の4つです。

5x

正の項です。

8

正の定数項です。

-2x

負の項です。2x ではなく、-2x で一つの項です。

7

正の定数項です。

つまり、式を整理するときは、数字や文字だけでなく、前に付いている + や - も一緒に見る必要があります。

Example

例題1：同類項をまとめる

例題 1
3x + 5 - 2x + 4 を簡単にしなさい。
1 3x + 5 - 2x + 4
2 = 3x - 2x + 5 + 4
3 = (3 - 2)x + 9
4 = x + 9

2行目では、x の項どうし、数字だけの項どうしを近くに並べ替えています。 このとき、-2x のマイナスを必ず一緒に動かすことが大事です。

Example

例題2：まとめてはいけない項を見分ける

例題 2
2x + 3y + 4x を簡単にしなさい。
1 2x + 3y + 4x
2 = 2x + 4x + 3y
3 = 6x + 3y

3y は x の項ではないので、6x とまとめません。 答えは 6x + 3y のままです。

Mistakes

間違えやすい点

x + x = x2 としてしまう

これは間違いです。x を2個足しているので、正しくは 2x です。 x2 は x × x の意味です。

2x + 3y = 5xy としてしまう

これも間違いです。x と y は違う種類なので、足して一つにはできません。 そのまま 2x + 3y と書きます。

5x - 2x を 7x にしてしまう

-2x のマイナスを見落としています。 5x - 2x は、係数で見ると 5 - 2 なので 3x です。

定数項を忘れる

3x + 5 + 2x + 4 を 5x だけで終わらせてはいけません。 5 + 4 もまとめて、答えは 5x + 9 です。

答えを表示

2 = 3a - a + 6 + 2

練習 4

Column

コラム：文字は「量の種類」を保つ札

ゼド先生の余談

式の中の文字は、ただの飾りではない。 x や y は、その量が何を表しているかを保つための札だ。 だから、x の項と y の項を勝手に混ぜると、何の量を扱っていたのか分からなくなる。 数学で式を整理するとは、見た目を短くすることではなく、意味を壊さずに整えることだ。

Summary

今日のまとめ

  • 項とは、+ や - で区切られた式の部品である。
  • 係数とは、文字にかかっている数字である。
  • 定数項とは、文字を含まない数字だけの項である。
  • 同類項とは、文字の部分が完全に同じ項である。
  • x と x2 は同類項ではない。
  • 符号は項の一部として読む。
  • 式を整理するときは、意味を壊さず、同じ種類だけをまとめる。

# キャラクター画像の使い分け

統合ルールに合わせて、画像は用途別に分けます。

```txt
zed/bust/      吹き出し・短評・コラム用のバストアップ
zed/standing/  人物紹介・ホーム・教科導入・章導入用の立ち絵
zed/action/    動きのある補助イラスト

mia/bust/      吹き出し・つまずき・コラム用のバストアップ
mia/standing/  人物紹介・ホーム・教科導入・章導入用の立ち絵
mia/action/    動きのある補助イラスト
```

立ち絵は吹き出しアイコンとして使いません。
吹き出しや短い補足には `.character-bust`、人物紹介や導入には `.character-standing` を使います。

## 1. このHTML例で参照するポイント

このv4プレビューは、以下の方針を実装例として確認するためのもの。

```txt
・三桁番号フォルダ構成
・複数ページ型サイト
・ホームは凝った入口ページ
・ホームでは更新情報、日誌、教科選択、人物紹介を中心にする
・数学偏重のホーム導線を置かない
・講義ページは右側固定レールなし
・本文幅を広く取る
・分野ページは本の目次風
・章の下に講義小目次を置く
・章ページ内で複数講義を切り替える
・#lesson-001 形式で講義へ直接リンクする
・練習問題の答えは折りたたみ式
・確認問題は任意開閉式
・確認問題の答えも折りたたみ式
・解答には途中式を入れる
・コラムはキャラ一人＋イラスト＋吹き出し形式
・数学表記は x^2 ではなく上付き表示を使う
```

## 6. 今後HTML生成時に守ること

このHTML例を参照して、今後のページ生成では以下を守る。

```txt
1. index.html はホームとして残す
2. フォルダ・章ファイルは三桁番号を使う
3. 教科 → 分野 → 章 → 講義小目次の階層を保つ
4. 章ページ内の講義IDは lesson-001 形式にする
5. 章ページ内の直接リンクは #lesson-001 形式にする
6. 練習問題の答えは最初から見せない
7. 確認問題は任意で開く
8. 確認問題の解答も任意で開く
9. 解答には途中式を入れる
10. コラムはキャラ一人＋イラスト＋吹き出し形式にする
11. 数学の二乗は x^2 ではなく x<sup>2</sup> を使う
12. 右側固定レールは作らない
13. 未実装機能を目立つボタンとして置かない
```

## 統合ルール反映メモ

- 教科順は哲学、数学、国語、理科、社会。
- 各教科は `000_syllabus.html` から制作する。
- 内部管理は三桁番号、画面表示は `第1章` / `第1講`。
- バストアップ画像は吹き出し・短評・コラム用。
- 立ち絵画像は人物紹介・ホーム・教科導入・章導入用。
